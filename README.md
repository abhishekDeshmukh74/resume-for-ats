# pass-ats — AI Resume Tailor

Upload your resume (PDF/DOCX), provide a job description (paste or URL), and get an ATS-optimised resume generated by Groq AI. Download it as a clean PDF.

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + Vite (TypeScript) + Tailwind CSS |
| Backend | Python + FastAPI |
| AI | Groq API (`llama-3.3-70b-versatile`) |
| PDF | ReportLab |

---

## Setup

### Prerequisites
- Python 3.11+
- Node.js 20+
- A free [Groq API key](https://console.groq.com/)

### 1. Clone & configure

```bash
cd pass-ats
cp .env.example .env
# Edit .env and set your GROQ_API_KEY
```

### 2. Backend

```bash
cd backend
python -m venv .venv

# Windows
.venv\Scripts\activate

# macOS/Linux
source .venv/bin/activate

pip install -r requirements.txt
```

### 3. Frontend

```bash
cd frontend
npm install
```

---

## Running

Open **two terminals**:

**Terminal 1 — Backend**
```bash
cd e:\code\pass-ats
.venv\Scripts\activate   # or source .venv/bin/activate
uvicorn backend.main:app --reload --port 8000
```

**Terminal 2 — Frontend**
```bash
cd e:\code\pass-ats\frontend
npm run dev
```

Then open http://localhost:5173 in your browser.

## How It Works

1. **Step 1 — Upload Resume**: Drag-drop or select a `.pdf` or `.docx` file. The backend extracts the text.
2. **Step 2 — Job Description**: Paste the JD text, or provide a URL and the backend scrapes it.
3. **Step 3 — Generate**: Groq AI rewrites your resume to match the JD, mirroring keywords and phrasing for ATS optimisation.
4. **Step 4 — Preview & Download**: Review the tailored resume on-screen and download it as a clean single-column PDF.

## API Endpoints

| Method | Path | Description |
|--------|------|-------------|
| POST | `/api/parse-resume` | Upload PDF/DOCX, returns extracted text |
| POST | `/api/scrape-jd` | `{url}` → returns scraped JD text |
| POST | `/api/generate-resume` | `{resume_text, jd_text}` → returns `ResumeData` JSON |
| POST | `/api/download-pdf` | `ResumeData` body → returns PDF stream |
| GET | `/api/health` | Health check |
